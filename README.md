# CSV-to-Adobe Illustrator Automation
Português (PT-BR)

Script que consome dados da API da Tray em formato CSV, identifica automaticamente o modelo e o tipo de grafia de cada item e reproduz as informações no Adobe Illustrator. O processo preenche todos os campos necessários e organiza os arquivos gerados de acordo com o modelo correspondente.

English (EN)

Script that consumes CSV data from the Tray API, automatically identifies the model and lettering type of each item, and transfers the information to Adobe Illustrator. The process fills in all required fields and organizes the generated files according to their corresponding model.
